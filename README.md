# Portfolio Website

A modern, responsive portfolio website showcasing my projects, skills, and experience.

## About

Personal portfolio built with Node.js, Express, and Tailwind CSS featuring smooth animations and an intuitive user interface.

## Tech Stack

- Node.js
- Express.js
- EJS (Embedded JavaScript templating)
- Tailwind CSS
- AOS (Animate On Scroll)
- Nodemailer (Contact form)

## Features

- Responsive design
- Smooth scroll animations
- Contact form with email integration
- Project showcase
- GitHub contributions display
- Dark theme

## Projects Included

1. College Ride Share - Full-stack ride-sharing platform
2. Driver Drowsiness Detection - AI-powered safety system
3. Portfolio Website - This website

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Create .env file with your email credentials
4. Run the server: `npm start`
5. Open browser at: `http://localhost:3000`

## Environment Variables
```
PORT=3000
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-password
EMAIL_TO=your-email@gmail.com
```
